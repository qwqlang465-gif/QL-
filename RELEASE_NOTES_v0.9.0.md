# QL v0.9.0

这版重点优化手机端 APK 的阅读体验，让 QL 更接近个人常用的移动小说阅读器。

## 新增

- 手机端阅读页支持左右点击翻页和横向滑动翻页。
- 底部栏调整为目录、界面、设置、日间/夜间切换。
- 界面面板新增覆盖、滑动、仿真、滚动、无动画翻页模式。
- 界面面板新增字号、字重、字距、行距、段距、两端对齐、底部对齐。
- 字体新增系统、黑体、宋体、楷体、等宽。
- 主题新增暖阳、雾蓝、夜间模式。
- APK 新增屏幕常亮、扩展到刘海、隐藏状态栏、隐藏导航栏、音量键翻页。

## 调整

- 手机端顶部栏进一步简化，减少阅读干扰。
- 设置入口收纳搜索、书签、加书签、章节切换、返回书架和沉浸式控制。
- 默认字体改为更适合手机中文阅读的系统字体栈。

## 验证

- 已通过 `npm run build`。
- 已通过 `npm run cap:sync`。
- 已通过 `npm run desktop:pack`。
- Android APK 原生编译由 GitHub Actions 使用 JDK 21 环境完成。
