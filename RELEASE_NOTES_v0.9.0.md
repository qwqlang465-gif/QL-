# QL v0.9.0

这一版主要优化手机端阅读体验。

## 下载

- 手机端：`QL-v0.9.0-release.apk`
- 电脑端：`QL-v0.9.0-win-unpacked.zip`

## 重点更新

- 手机端支持左右点击翻页、横向滑动翻页。
- 底部栏改为目录、界面、设置、日间/夜间。
- 新增音量键翻页。
- 新增屏幕常亮、扩展到刘海、隐藏状态栏、隐藏导航栏。
- 新增覆盖、滑动、仿真、滚动、无动画翻页。
- 新增字号、字重、字距、行距、段距。
- 新增两端对齐、底部对齐。
- 新增系统、黑体、宋体、楷体、等宽字体。
- 新增暖阳、雾蓝、夜间主题。

## 验证

- `npm run build`
- `npm run cap:sync`
- `npm run desktop:pack`
- GitHub Actions Android APK 构建成功
- GitHub Actions Windows exe 构建成功
